貢献・開発参加ガイド

このBotはオープンソースです！どなたでも貢献・修正・機能追加を歓迎します 🎉

貢献の流れ

  1. GitHubで このリポジトリをFork
  2. 新しいブランチをつくって変更・開発
  3. コミット＆ Pull Request（PR）を送信
  4. レビュー・マージ後、みんなでアップデートをシェア！

Issueや要望・バグ報告

  • 「こんなコマンドがほしい」「バグ発見！」など、お気軽に Issue をご活用ください。
  • テンプレートやGitHub Discussionsも適時利用！

開発のヒント

  • 新コマンド追加は bot.py の @tree.command デコレータを利用
  • ドキュメント・Wikiは随時修正・追記大歓迎
  • テスト・リファクタ・コードコメント強化なども◎！
