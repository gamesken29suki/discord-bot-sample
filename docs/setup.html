セットアップガイド

このページでは、Discord Bot Sample の導入から起動までの手順をわかりやすく解説します。

  1. GitHubリポジトリをクローン/ダウンロード
    https://github.com/gamesken29suki/discord-bot-sample
  2. Discord Botの登録 & Token取得
    Discord Developer Portal で新規アプリ追加→Bot登録→Token発行
    ※Tokenは絶対に他人に見せないでください！
  3. .envファイルを作成し、下記のように記述
    DISCORD_BOT_TOKEN=ここに取得したトークン
  4. Pythonパッケージをインストール
    pip install -U discord.py python-dotenv
  5. Botをサーバーに招待
    OAuth2のURL Generatorで「bot」「applications.commands」権限を選んでURL発行
    サーバーへBotを招待
  6. bot.py を実行
    python bot.py
    コマンドが「同期完了」となれば成功！
ヒント: くわしい手順やトラブル対応は FAQ もチェック！
トークンや.envはGitなどで絶対に公開しないようご注意を。
