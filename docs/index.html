Discord Bot Sample

GitHub で見る 📚 使い方を見る サポートサーバー へ参加

このBotは？

Python と discord.py（ver.2.x）で動作するサンプルBot！
教育・学習・改造・カスタムコマンドの教材にも最適です。

💬 チャット例

ユーザー: /hello
Bot: こんにちは！ 👋
ユーザー: /fortune
Bot: 🎋 今日の運勢は「大吉」です！
ユーザー: /dice 6
Bot: 🎲 サイコロの結果: 4

🤖 主なコマンド

👋 あいさつ
/hello

基本的なあいさつコマンド

🎋 おみくじ
/fortune

今日の運勢を占います

🎲 サイコロ
/dice

ランダムな数値を生成

✂️ じゃんけん
/janken

Botとじゃんけん勝負

👤 ユーザー情報
/userinfo

ユーザーの詳細情報を表示

⏰ タイマー
/timer

時間を測定します

🎯 多面ダイス
/roll NdM 目標

多面ダイスや判定も対応 | 全コマンドを見る →

bot.pyに@tree.commandの形で新コマンドを追加し、Discordでコマンド同期すればOKです。Wiki・READMEにサンプル実装があります。

GitHubページやWikiのセットアップガイドの通り、Bot登録＆Token取得→.env→依存モジュールインストール→Bot起動で使えます！
