API・技術解説

このページでは、Discord Bot Sample の内部構造や拡張API、開発者向けの技術情報を解説します。

コマンド追加・カスタマイズ方法（サンプル）


@tree.command(name="sample", description="サンプルコマンド")
async def sample_command(interaction: discord.Interaction):
    await interaction.response.send_message("これはサンプルの応答です")

主な拡張ポイント

  • コマンド追加: @tree.command デコレータを利用
  • Botイベント登録: @client.event でon_ready/on_message等を拡張
  • 環境変数: python-dotenv で.env取得
  • メンバー権限制御: ロール判定や限定コマンドの実装

使っている技術・ライブラリ

  • discord.py（2.x系）
  • python-dotenv
  • Python 3.8以降
  • GitHub Actions（CI/CD 自動テスト等、今後導入予定）
